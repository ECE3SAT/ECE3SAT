The NucleoCode folder gives the code to be able to use the nucleo Board
The ProcessingCode provides the visualisation and computation of the data we read on the board

Be sure to read the "Readme" of the folders to make it work

How to install the project :

-Install the libraries (look at the instructions in the following folders)
-Copy the folder "Orientation_Sensors_Reading" on the computer

How to use it:

-Open the processing project "Orientation_Sensors_Reading" (folder ProcessingCode)
-Plug the Nucleo on the sub and put the binary inside (folder Nucleo code, put it twice)
-Launch the program with the play buttom
